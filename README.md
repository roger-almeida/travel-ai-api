# Travel AI API

API REST de Corporate Travel Assistant com IA integrada.

Projeto construído como parte de um plano intensivo de estudos para vaga sênior com stack Java + Cloud + LLMs.

## Stack

- **Java 21** + Spring Boot
- **Docker** + Kubernetes (minikube)
- **AWS** DynamoDB
- **LLM** (Anthropic Claude / OpenAI)
- **RAG** com busca vetorial
